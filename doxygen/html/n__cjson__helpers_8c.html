)

Decode a Base64-encoded string field in a JSON object and return the decoded bytes.

Parameters
jsonThe JSON object to query.
fieldNameThe field name to retrieve.
retBinaryDataA pointer to a pointer used to store the decoded binary data (caller must free).
retBinaryDataLenA pointer to an unsigned integer used to store the length of the decoded binary data.
Returns
true if the binary data was successfully decoded, false otherwise.
Note
The returned binary buffer must be freed by the user with JFree when it is no longer needed.
On error, the returned binary buffer and data length shall be set to NULL and zero (0), respectively.
